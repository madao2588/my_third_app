# 前端页面地图

## 一级导航

1. 首页看板
2. 公告中心
3. 关键词规则
4. 来源网站
5. 系统管理

## 页面说明

### 首页看板

目标：
- 让用户第一眼看到当天是否有重要招标信息

主要区域：
- 核心指标卡
- 高价值公告
- 关键词热度
- 来源网站分布
- 最新动态

### 公告中心

目标：
- 作为日常审阅公告的主工作台

主要区域：
- 筛选栏
- 公告列表
- 公告详情
- 原文跳转按钮

### 关键词规则

目标：
- 维护关键词和优先级

主要区域：
- 分组列表
- 关键词列表
- 启用状态
- 优先级设置

### 来源网站

目标：
- 管理监测站点

主要区域：
- 网站列表
- 网站详情
- 抓取方式
- 最近抓取结果

### 系统管理

目标：
- 提供任务、日志、调度和异常查看能力

主要区域：
- 监测任务
- 调度状态
- 运行日志
- 异常记录

## 主流程

1. 从首页看板进入系统
2. 从首页的高价值公告或快捷筛选进入公告中心
3. 在公告中心筛选并查看详情
4. 通过原文按钮打开源站页面
5. 如需调整规则，再进入关键词规则或来源网站页
