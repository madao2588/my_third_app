# 低保真线框说明

## 首页看板

```text
+------------------------------------------------------------------+
| 侧边导航 | 顶部栏：首页看板                     更新时间   刷新   |
+------------------------------------------------------------------+
| 指标卡1：今日新增公告 | 指标卡2：命中关键词公告 | 指标卡3：监测站点 |
+------------------------------------------------------------------+
| 最新高价值公告 Top 5              | 关键词热度 / 词云             |
| - 标题                            | - 招标                        |
| - 地区                            | - 原料药                      |
| - 发布时间                        | - 注射液                      |
+------------------------------------------------------------------+
| 来源网站分布图                    | 来源网站统计表               |
+------------------------------------------------------------------+
```

首页重点：
- 强调今天发生了什么
- 强调哪些公告值得看
- 强调关键词和来源分布

## 公告中心

```text
+------------------------------------------------------------------------+
| 侧边导航 | 顶部栏：公告中心                                            |
+------------------------------------------------------------------------+
| 搜索 | 时间范围 | 来源网站 | 公告类型 | 是否重点 | 是否已读 | 重置     |
+------------------------------------------------------------------------+
| 公告列表                                        | 详情区                |
| ------------------------------------------------| 标题                  |
| 标题                                            | 发布时间              |
| 来源网站                                        | 来源网站              |
| 发布时间  关键词  优先级                        | 命中关键词            |
| [查看] [打开原文] [标记重点]                    | 正文                  |
| ------------------------------------------------| 快照入口              |
| 下一条...                                       | 打开原文按钮          |
+------------------------------------------------------------------------+
```

公告中心重点：
- 筛选要快
- 标题要醒目
- 原文入口要显眼
- 详情区要适合连续审阅

## 设计提醒

- 页面文案尽量使用业务词汇，不直接暴露后端技术名称
- 原文跳转按钮必须在详情区保持可见
- 首版优先确保结构清楚，不必先做复杂动画
